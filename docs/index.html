Wu

A low-level backtesting library for trading strategies written in C

Repository API Reference Tutorial

Wu is a toolkit, not a framework. The core abstractions compose freely. You can swap implementations, combine strategies, or bypass components without constraints. Built for learning, experimenting with different designs, and exploring ideas in algorithmic trading.

Portfolios

Pluggable strategy interface for signal generation

Quick Start

make
make run_tests
sudo make install

Data Types

Wu supports three data types:

  • Candles (OHLCV): Standard bar data
  • Trades: Tick-level data with price, volume, and side
  • Single Values: Generic time-series data

Design Philosophy

The library uses C's struct-and-function-pointer pattern for polymorphism. Each abstraction – Portfolio, Strategy, Reader, and Indicator – is a struct with function pointers, allowing different implementations with a consistent interface.
